# Release 1.2.3

## Bundle Production Release

Release CR used to release the bundle to the production registry.

```yaml
apiVersion: appstudio.redhat.com/v1alpha1
kind: Release
metadata:
  name: ext-dns-optr-1-2-3
  namespace: external-dns-operator-tenant
  labels:
    release.appstudio.openshift.io/automated: "false"
    release.appstudio.openshift.io/author: dsalerno
spec:
  releasePlan: external-dns-operator-1-2-release-plan-prod
  snapshot: ext-dns-optr-1-2-rhel-8-20260622-091709-000
```
