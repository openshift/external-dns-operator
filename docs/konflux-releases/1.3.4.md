# Release 1.3.4

Jira: [OCPBUGS-79591](https://issues.redhat.com/browse/OCPBUGS-79591)

This patch release includes a Golang bump to v1.25.8, base image updates, and Konflux label fixes (`name`, `cpe`) to match product security metadata.
